Border Radius

1 valor:

Aplica para todos os lados:

2 valores:

  1. Aplica para o topo esquerdo e o canto inferior direito.
  2. Aplica para o topo direto e canto inferior esquerdo

3 valores:

  1. Aplica o topo esquerdo.
  2. Aplica o topo direito e o canto inferior esquerdo.
  3. Aplica o canto inferior direito

4 valores:

  1. Aplica para o topo esquerdo.
  2. Aplica para o topo direito.
  3. Aplica para o canto inf direito.
  4. Aplica para o canto inf esquerdo.

Para o círculo, aplicamos um border-radius de 50%, mas é necessário que a altura e largura sejam iguais.
