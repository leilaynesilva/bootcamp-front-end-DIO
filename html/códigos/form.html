Formulários:

Formularios são campos onde o usuario insere informações, é relacionado com o servidor.

Quando clica em enviar, transmite as informações pro servidor

Nome:
Idade:
Senha:

O código do formulário fica assim:

<form onsubmit="alert('Esse formulário já foi enviado')">
Nome: <input type="text" name="name"/><br>
Idade: <input type="number"name="age"><<br>
Senha: <input type="password"name="password"/><br>
<button type="submit">Enviar</button>
</form>
  • o input é filho de form e é um campo de inserir valores.
  • on alguma coisa representa um evento em javascript

Atributos de form:

  • name: é o nome do formulário (mais usado para o javascript)
  • action: pra onde os dados vão, onde coloco o endereço do servidor.
    ex.: <form action= "sitedoservidor.com"></form>
  • method: como as informações serão enviadas ao servidor:
    • GET: manda as info por URL, n é indicado p senhas.
    • POST: manda as info p requisição, enpacota a informação e só desenpacota lá no servidor.
  • target: controla em que aba vai executar
    • blank: abre o conteudo em guia diferente.
    • self: abre o conteudo na mesma guia.
  • autocomplete: completa as informações do usuario automaticamente. Se ON permite essa função, se OFF se encontra desligada.
  • on alguma coisa:
    • significa evento
    • quando é onsubmit quando clica em cima aciona uma ação javascript, geralmente valida o formulário.
