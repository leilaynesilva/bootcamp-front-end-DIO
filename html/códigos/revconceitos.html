Revisando conceitos vistos até aqui

Estrutura básica do HTML:

  • head: onde ficam os códigos que são executados antes do que o usuário vai ver. São os bastidores
  • body: onde ficam as coisas visíveis ao usuário.

Tags:

  • tag i = transforma o texto em itálico
  • tag strong = transforma o textoem negrito.

Tags automáticas:

  • tag input.
  • tag img.

Atributos:

  • id = identificação.
  • style = comandos CSS dentro do html.
  • class = grupo de estilo no CSS.

Atributos específicos:

  • tag input:
    • type = especifica o tipo de caractere de entrada(text , number, etc)
  • tag img:
    • src= "" define o caminho do arquivo.
    • widht = "" define a largura da imagem.
    • height = "" define a altura da imagem.

Textos

São as tags usadas na tipografia, como os "h" (h1, h2,h3, etc) e o p.

  • blockquote = citação / observação.
  • i = italico.
  • mark = marca-texto.
  • sup = elevado.
  • sub = letrinha pequena.

Listas

São as listas ordenadas (ol) e as não ordenadas (ul)

Onde ol são as que tem uma ordem (1,2,3,4) e a ul são marcadores

    Exemplo de "ol":
  1. Acordar
  2. Tomar café
  3. Escovar os dentes
    Exemplo de "ul":
  • Comprar arroz.
  • Comprar feijão.
  • Comprar carne.

Links

Os links direcionam para outras páginas, sejam dentro do meu site ou externo.

Tag anchor (a)

Ex. <a href = "endereçodosite.com>Aqui coloco o texto que vai aparecer</a>

Atributos:

  • target blank = abre o link em guia diferente.
  • target self = abre na mesma guia.
  • title = define um txt que vai aparecer quando passar o mouse.
