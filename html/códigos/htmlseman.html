Tags Semanticas

Tag header

Cabeçalho

  • Header pode ficar dentro de main.

Tag main

O assunto principal da página.

  • Só pode ter um no html todo, não pode ser filha de nenhuma outra tag de estrutura (header, footer, article, nav, etc.)

Tag section

É usada para dar o resultado de buscas, por exemplo, como se fosse a div mas é mais semântico.
Ex.: quando faço uma busca de casas de aluguel, o resultado dessa busca será exibido dentro de tags section.

Tag Nav

Usada para menus, navegação.
Geralmente vem acompanhada de tags ul.
É onde colocamos os links.

Tag article

Usada para artigos/ conteúdo relacionado ao site.
